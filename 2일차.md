#### 형변환(Casting)
1. __숫자__ 를 다른 형태의 __숫자__ 로 변환
2. __다른 자료형__ 을 __문자__ 로 변환
3. __문자__ 를 __다른 자료형__ 으로 변환

##### 숫자를 다른 숫자의 형태로 변환(int, float)

  * float → int <br>
  
        int x = 10;   
        float y = 10.0f;
     
        int z = x +(int)y;

  * int → float <br>
  
        int x = 10;   
        float y = 10.0f;
   
        float z = (float)x + y;

> 데이터의 손실이 없는 경우 괄호로 형태를 명시하지 않아도 캐스팅이 가능하다.<br>
> float 1.5를 int 1로 바꾸려면 0.5의 값이 사라지기 때문에 명시해야 하지만,<br> int 1을 float 1.0f로 바꾸려면 누락될 값이 없기 때문에 명시하지 않아도 된다.
> 
>     int x = 10;
>     float y = 10.0f;
>     
>     float z = x + y;

#### 변수의 규칙 - 생성과 저장
* (변수)생성 : 변수 앞에 __데이터 형식__ 이 있다.
  > __string__ Chocolate = "ABC"; //변수의 생성 후 데이터 저장
 
* (값)저장 : 변수 앞에 데이터 형식이 없다.
  > Chocolate = "ABC"; //이미 만들어진 변수에 데이터만 저장

### 연산자
__연산자__ 란 변수나 값에 대해 계산 또는 조작을 수행하는 기호나 키워드를 의미한다.
#### 산술 연산 _//사칙 연산_
기호|내용
|-|-|
+|더하기
-|빼기
*|곱하기
/|나누기
%|나머지
++|증가
--|감소

* 더하기 연산(+)
  
        int x = 10;   
        int result;
  
        result = x + 3;   
        Console.WriteLine(result); //13
* 빼기 연산(-)
  
        int x = 10;
        int result;

        result = x - 5; //result : 5
* 곱하기(*)

        int x = 10;
        int result;

        result = x * 2; //result : 20
* 나누기, 나머지 연산(/,%)
  
        int x = 17;
        int result_1;
        int result_2;

        result_1 = x / 5; //result_1 : 3 (몫)
        result_2 = x % 5; //result_2 : 2 (나머지값)

  #### Tip! / 연산보다 * 연산이 더 빠르므로, * 연산 이용 권장
        x = 10 / 2   
        x = 10 * 0.5f //이 계산식이 더 빠르게 처리됨.

  * 증감 연산(++,--)
    > __1씩 연산__ <br>
    > 증감은 1씩 계산된다.
    >
    >      int x = 10;   
    >      x++; // x  :11
    >
    >      int y = 10;   
    >      y--; // y : 9

    > __변수에 바로 계산__ <br>
    > 일반적으로 변수는 연산을 한 값을 다시 할당하지 않으면 저장되지 않는다.
    > 
    >      int x = 10;
    >
    >     Console.WriteLine(x + 10); // 20 출력
    >     Console.WriteLine(x); // 10 출력
    > x에 10을 더했지만 다시 할당하지 않았으므로 x는 그대로 10이다.
    > 
    >      int x = 10;
    >      x = x + 10; // =을 이용해 x에 다시 값을 할당하였으므로
    >
    >      Console.WriteLine(x); // 20 출력
    >
    > 이렇게 일반 연산(+,-,*,/,%)는 다시 할당을 해야 값이 저장된다.<br>
    > 하지만 증감 연산자는 사용 후 다시 할당하지 않더라도 값이 저장된다.
    > 
    >      int x = 10;
    >      x++;
    > 
    >      Console.WriteLine(x); // 11 출력
    >
    >      int y = 10;
    >      y--;
    > 
    >      Console.WriteLine(y); // 9 출력
     
    > __전위 연산/후위 연산__ <br>
    > 증감 연산은 해당 연산을 변수의 앞/뒤에 붙일 수 있다.
    > 
    >     int x = 10;
    >     x++;
    >     ++x;
    > 
    > 세미콜론(;)을 기준으로 띄어쓰기를 하며 이를 각 __라인__ 이라 한다.<br>
    > C#에서는 라인을 기준으로 한 라인씩 작업을 수행하는데,<br>
    > 연산자를 __뒤__ 에 붙이면 해당 라인이 __끝나고 값이 증가__ 한다.<br>
    > 연산자를 __앞__ 에 붙이면 해당 라인을 __실행할 때 증가된 값으로 실행__ 한다.<br>
    
    >     int x = 10;
    >     Console.WriteLine(x++); // ++를 x뒤에 붙였기 때문에 아직 10이 출력된다.
    >     Console.WriteLine(++x); // ++를 x 앞에 붙였기 때문에 11로 출력된다.
    >
    >     int y = 10;
    >     Console.WriteLine(++y); // 11 출력
    >     Console.WriteLine(y); // y에 11이 저장되었기 때문에 11로 출력된다.

   * x = x + 1;을 간소화하면,
     
          int x = 10;
          x = x + 10;
          Debug.Log(x) // 20

          int y = 10;
          y += 10;
          Debug.Log(y) // 20

          (+,-,*,/,%) 모두 가능하며 a += 10;의 형태로 사용한다.
